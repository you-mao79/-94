 ##[资源集成]：一个基于 [Spring Boot + Vue] 的资源管理系统

## ✨ 核心功能场景

*   **场景一：资源全流程管理**：用户可以上传、在线预览、分类筛选、储存和导出资源。
*   **场景二：个人工作区**：提供“我的收藏”、“浏览历史”、“个人设置” 会根据不同类型来设置开源“工具模块”。
*   **场景三：权限控制**：区分普通用户与管理员，后台支持用户管理、角色分配。

## 🛠️ 技术栈
*   **后端**：Spring Boot 2.x, MyBatis-Plus, MySQL, JWT
*   **前端**：Vue 3, Element Plus, Axios, Vue Router

## 🗂️ 项目亮点

*   **亮点 1: [例如：高效的企业级权限管理]**：基于 RBAC 模型，通过注解 + 拦截器实现了细粒度的接口权限控制，保证了系统安全。
*   **亮点 2: [例如：流畅的文件上传体验]**：实现大文件分片上传与断点续传，进行文件加速存储。
*   **亮点 3: [例如：手写核心解析引擎]**：独立设计并实现了 `[资源元数据解析模块]`，完成了对 `[例如：30+]` 种资源类型的自动识别和元数据提取。

## 🚀 快速体验

1.  **克隆项目**：`git clone [你的仓库地址]`
2.  **导入数据库**：新建 `[数据库名]` 库，运行 `db/schema.sql` 完成初始化。
3.  **修改配置**：打开 `application.yml`，填入你的数据库用户名和密码。
4.  **启动后端**：运行 `ZomApplication.java` 的主方法。
5.  **访问项目**：浏览器打开 `http://localhost:8091`
